Sistema de Archivos y Almacenamiento Secundario

Organización y gestión eficiente de datos

El sistema de archivos organiza y gestiona los datos en el almacenamiento secundario (discos duros, SSDs, USBs).

Conceptos clave:

  • Este almacenamiento es no volátil (mantiene los datos sin energía)
  • Tiene una mayor capacidad que la memoria RAM.
  • El sistema de archivos controla cómo se guardan, nombran y estructuran (en carpetas) los archivos.

Esto permite la recuperación de los datos de manera eficiente cuando el sistema se enciende.
