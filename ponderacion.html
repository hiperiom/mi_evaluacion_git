Criterios de Evaluación (Escala 20 pts)

La siguiente rúbrica detalla los parámetros evaluados en la práctica colaborativa de Git y HTML.

Criterio Descripción esperada Puntaje
1. Flujo de Trabajo Git Clonado exitoso del repositorio. Creación de una rama local aislada con el formato feature/nombre-tema y subida (push) correcta usando el rastreo remoto (-u origin). 5 pts
2. Estructura y Clean Code (HTML) Página en paginas_estudiantes/ usando la plantilla proporcionada y siguiendo buenas prácticas de HTML. 5 pts
3. Historial de Commits Realización de 5 commits atómicos. Los mensajes de commit descriptivos que se indican en las instrucciones. 5 pts
4. Navegación y Enlaces (Rutas) Modificación de index.html (enlace relativo) y retorno desde su página hacia la raíz. 5 pts
